# GameToken DApp - Base Builder Edition

A high-performance decentralized application built on **Base Mainnet**. This app allows users to manage their assets, perform fast transfers, and swap tokens via Uniswap while ensuring correct builder attribution for the Base ecosystem.

## 🚀 Features

- **Multi-Asset Support**: Real-time balance tracking for ETH and USDC on Base.
- **Fast Transfers**: Send ETH or USDC to any address with optimized gas strategy.
- **Uniswap Integration**: Seamless token swaps directly through the Uniswap V3 Router.
- **Builder Attribution**: Every transaction is marked with a unique **Builder Code** (`0x62635f...`) to support the Base Builders & Founders initiative.
- **Network Toggle**: Support for both Base Mainnet and Base Sepolia testnet.

## 🛠 Tech Stack

- **Frontend**: React + Vite
- **Blockchain Library**: Ethers.js
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Deployment**: Vercel

## 🛡 Builder Integrity

This project is part of the Base ecosystem. To verify our transactions on the Base Dashboard, we use custom calldata appending for all smart contract interactions.
