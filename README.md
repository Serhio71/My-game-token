# GameToken DApp - Base Builder Edition

A high-performance decentralized application built on **Base Mainnet**.

## 🚀 Features
- **Builder Attribution**: Every transaction is marked with a unique Builder Code.
- **Uniswap V3**: Integrated swapping engine.
- **Network**: Operations on Base Mainnet and Base Sepolia.
